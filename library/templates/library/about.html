Meet the A22 Team

We are students from the School of Engineering and Applied Science at the University of Virginia, building this library management system as part of our coursework.

About the Project

This library management system was developed as part of a course project at the University of Virginia's School of Engineering and Applied Science. Our goal was to create a functional and user-friendly application that demonstrates software engineering principles.

Technologies Used:
  • Django (Python web framework)
  • Bootstrap 5 (Frontend framework)
  • SQLite (Database)
  • JavaScript (Client-side functionality)
Key Features:
  • Book catalog management
  • User account system
  • Book lending and tracking
  • Collections for organizing books
  • Responsive design for all devices

Note: This project is for educational purposes only. The books and information shown are not real and are used solely for demonstration.
